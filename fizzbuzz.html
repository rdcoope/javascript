Here is a thought process using the "FizzBuzz" technique

In simple terms, FizzBuzz is a simple number replacement where numbers divisible by 3 are replaced with Fizz and numbers divisible by 5 are replaced with Buzz. Any numbers divisible by both are replaced by FizzBuzz.

Here is my Pseudo-Code for this

  1. generate a list of numbers from 1 to x (lets say x is 100 for now)
  2. itterate through the list
  3. Check if each number is divisible by 3
  4. Replace 3 with Fizz
  5. Check if number is divisible by 5
  6. Replace 5 with Buzz
    • if the number is divisible by 5 see if the number is divisible by 3
    • check if divisible by 3 and 5
    • if both are true, replace number with FizzBuzz
